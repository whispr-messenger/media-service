# Contribuer au Media Service

## Prérequis

- Node.js 18+
- Docker & Docker Compose
- PostgreSQL 15+
- Redis 7+

## Lancer le projet

```bash
npm install
npm run start:dev
```

## Conventions

- Conventional commits (`feat`, `fix`, `docs`, `test`)
- Branches : `WHISPR-XXX-description`
- Tests avant chaque PR
