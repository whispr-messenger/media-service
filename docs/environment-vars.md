# Variables d'environnement

| Variable | Description |
|----------|-------------|
| PORT | Port HTTP (défaut 3000) |
| DATABASE_URL | URL PostgreSQL |
| REDIS_HOST | Hôte Redis |
| REDIS_PORT | Port Redis |
| S3_ENDPOINT | URL endpoint S3/MinIO |
| S3_BUCKET | Nom du bucket |
| S3_ACCESS_KEY_ID | Clé d'accès S3 |
| S3_SECRET_ACCESS_KEY | Clé secrète S3 |
| ENCRYPTION_KEY | Clé de chiffrement AES-256 |
| AUTH_SERVICE_URL | URL auth-service (JWKS) |
| MODERATION_SERVICE_URL | URL moderation-service (gRPC) |
