# Codes d'erreur

| Code | Message | Description |
|------|---------|-------------|
| 400 | Invalid file type | Type MIME non supporté |
| 400 | File too large | Fichier > 100 Mo |
| 401 | Unauthorized | Token invalide |
| 403 | Access denied | Pas les permissions |
| 404 | Media not found | Fichier introuvable |
| 413 | Quota exceeded | Quota de stockage dépassé |
| 500 | Upload failed | Erreur lors de l'upload |
